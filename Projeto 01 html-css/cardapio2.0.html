Detalhes

cappuccino
Cappuccino, do italiano, pronunciado: ou capuchino, geralmente pronunciado: em português, é uma bebida italiana preparada com café expresso e leite.
Irish coffee
Irish coffee é uma bebida a base de café, uísque irlandês, açúcar e chantilly.
espresso
Um café expresso, frequentemente referido simplesmente como expresso, é um método de preparar café através da passagem de água quente, não fervente, sob alta pressão pelo café moído.
caffè latte
Latte é uma bebida de café expresso com uma quantidade generosa de espuma de leite no topo.
macchiato
O macchiato é feito com café espresso.
